# Under-the-Stack: 웹 엔지니어링 학습 플랫폼

'Under-the-Stack'은 웹 개발자들이 추상화된 기술 너머의 로우 레벨 원리를 깊이 있게 이해할 수 있도록 돕는 인터랙티브 학습 플랫폼입니다.

## 🚀 프로젝트 개요
현대 웹 개발은 높은 수준의 프레임워크와 추상화 덕분에 편리해졌지만, 정작 그 이면에 있는 콜 스택, 힙 메모리, 이벤트 루프와 같은 동작 원리를 이해하는 데는 큰 간극이 있습니다. 본 프로젝트는 이러한 간극을 '데이터 중심의 시각화'를 통해 메우는 것을 목표로 합니다.

## 🛠 주요 기술 스택
- **Frontend**: React, TypeScript, TanStack Router/Query
- **Styling**: Tailwind CSS (v4), shadcn/ui
- **Animation**: Framer Motion
- **Quality Control**: Biome, Lefthook

## 🎨 주요 기능 (MVP)
1. **인터랙티브 시뮬레이터**: 코드의 실행 단계에 따라 동적으로 변화하는 메모리 상태(Stack, Queue, Heap)를 시각적으로 확인합니다.
2. **3분할 워크벤치**: 학습 내용 관찰, 가이드 확인, 시각화 도구를 한눈에 볼 수 있는 전문 지식 학습 환경을 제공합니다.
3. **선언적 상태 동기화**: URL 엔진을 통해 특정 학습 단계의 상태를 공유하고 북마크할 수 있습니다.

## 📂 프로젝트 구조
```text
src/
├── features/      # 도메인 기반 핵심 기능 (Simulator 등)
├── shared/        # 공통 UI 컴포넌트, 유틸리티, 전역 훅
├── routes/        # 페이지 및 라우팅 설정
└── styles/        # 전역 CSS 및 테마
```

## 📈 향후 로드맵
- MDX 기반의 체계적인 커리큘럼 엔진 도입
- Event Loop 및 JS 엔진 최적화 동작 시각화 모듈 추가
- 유저별 학습 진행 상태 저장 및 분석 기능
